Назначение.
Идея dw0774 Helper состоит в том, что при необходимости можно оставить заявку на какую-либо помощь, урегулировать вознаграждение. Заявка существует некоторое время. Другие пользователи могут откликнуться на Ваш зов. Помогли тебе - помоги другим.

Инструкция.
dw0774 Helper - переход на главную страницу.
Оставь заявку - переход к странице оформления заявки (только для зарегистрированных пользователей).
Помоги другому - переход к странице с заявками других пользователей.
Контакты - связь с разработчиком.

Технологии.
Front-end - Angular 7, Bootstrap 4.
Back-end - Java EE, Hibernate.
Сервер: Apache Tomcat.
База данных: MySQL.
Cборщик Maven.
REST архитектура.

Разработчик снимает с себя любую ответственность.
Сайт не адаптирован к просмотру с некоторых устройств.
